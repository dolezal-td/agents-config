# Checklist kontroly smlouvy

Projít každou sekci. U každého bodu zaznamenat: OK / problém / chybí.

---

## 1. Identifikace stran a základní náležitosti

- [ ] Jsou správně uvedeny obchodní firmy / jména obou stran?
- [ ] IČO, DIČ, sídlo – souhlasí s veřejnými rejstříky (OR, ARES)?
- [ ] Je uvedena osoba oprávněná jednat (statutár, zmocněnec)?
- [ ] Pokud jedná zmocněnec – je odkaz na plnou moc?
- [ ] Je smlouva datována?
- [ ] Je jasné, kdo je "objednatel"/"zhotovitel" (resp. jiné role)?

## 2. Předmět smlouvy

- [ ] Je předmět jednoznačně a konkrétně definován?
- [ ] Nejsou formulace příliš vágní ("a další služby", "dle potřeby")?
- [ ] Je jasný rozsah – co je zahrnuto a co nikoliv?
- [ ] Jsou definovány výstupy / deliverables (pokud relevantní)?
- [ ] Jsou specifikovány kvalitativní požadavky / akceptační kritéria?

## 3. Finanční podmínky

- [ ] Je cena/odměna jednoznačně stanovena (fixní / hodinová / milníková)?
- [ ] Je uvedeno, zda je cena s DPH nebo bez DPH?
- [ ] Jsou jasné platební podmínky (splatnost, způsob platby)?
- [ ] Jsou definovány podmínky pro změnu ceny (inflační doložka, eskalace)?
- [ ] Je řešeno, kdo nese náklady na cestovné, materiál, subdodávky?
- [ ] Jsou definovány sankce za pozdní platbu (úrok z prodlení)?
- [ ] Je stanoven postup pro fakturaci?
- [ ] Jsou jasné podmínky pro vícepráce / změnové požadavky?

## 4. Časové parametry

- [ ] Je stanoven začátek plnění?
- [ ] Je stanoven termín dokončení / doba trvání?
- [ ] Jsou definovány milníky (pokud relevantní)?
- [ ] Je jasné, zda jde o smlouvu na dobu určitou nebo neurčitou?
- [ ] Pokud na dobu určitou – je automatická prolongace? Za jakých podmínek?
- [ ] Jsou realistické lhůty pro jednotlivé kroky?

## 5. Odpovědnost a záruky

- [ ] Je definována odpovědnost za škodu a její limit (cap)?
- [ ] Je limit odpovědnosti symetrický pro obě strany?
- [ ] Jsou vyloučeny/omezeny následné a nepřímé škody?
- [ ] Je definována záruční doba?
- [ ] Je jasný postup pro reklamaci vad?
- [ ] Je řešena odpovědnost za škodu způsobenou třetím osobám?
- [ ] Je řešeno pojištění odpovědnosti (pokud relevantní)?

## 6. Sankce a smluvní pokuty

- [ ] Jsou smluvní pokuty přiměřené?
- [ ] Jsou pokuty symetrické (obě strany)?
- [ ] Je smluvní pokuta na úhradu škody nebo vedle škody?
- [ ] Nejsou kumulativní pokuty, které by v součtu byly nepřiměřené?
- [ ] Je definován strop (cap) na celkové smluvní pokuty?

## 7. Ukončení smlouvy

- [ ] Je definována výpovědní lhůta?
- [ ] Jsou vyjmenovány důvody pro výpověď bez výpovědní lhůty?
- [ ] Je možnost odstoupení od smlouvy (a za jakých podmínek)?
- [ ] Je řešeno vypořádání při ukončení (platby za odvedenou práci, vrácení materiálů)?
- [ ] Jsou definovány povinnosti po ukončení (přechodné období, migrace)?
- [ ] Je řešena force majeure?

## 8. Mlčenlivost a ochrana informací

- [ ] Je definován rozsah důvěrných informací?
- [ ] Není definice příliš široká (vše je důvěrné)?
- [ ] Jsou definovány výjimky (veřejně dostupné, nezávisle vyvinuté)?
- [ ] Je stanovena doba trvání mlčenlivosti?
- [ ] Je řešen postup při úniku informací?
- [ ] Je povinnost mlčenlivosti oboustranná?

## 9. Duševní vlastnictví (IP)

- [ ] Je jasně definováno, komu patří výstupy práce?
- [ ] Je řešen převod / licence k IP?
- [ ] Je definováno, co se děje s IP při ukončení smlouvy?
- [ ] Je řešeno předchozí IP (background IP) obou stran?
- [ ] Je jasné, zda může zhotovitel použít výstupy jako referenci?

## 10. Ochrana osobních údajů (GDPR)

- [ ] Je řešeno zpracování osobních údajů (pokud relevantní)?
- [ ] Je definován správce vs. zpracovatel?
- [ ] Existuje zpracovatelská smlouva (DPA) nebo odkaz na ni?
- [ ] Je řešen přenos údajů mimo EU/EHP?

## 11. Řešení sporů

- [ ] Je stanoveno rozhodné právo?
- [ ] Je určena příslušnost soudu?
- [ ] Je řešena rozhodčí doložka (pokud ano – je vyvážená)?
- [ ] Je povinný pokus o smírné řešení před soudním sporem?

## 12. Další ustanovení

- [ ] Je řešena možnost postoupení smlouvy (cessus)?
- [ ] Je definováno, jak se smlouva mění (písemné dodatky)?
- [ ] Je salvátorská klauzule (neplatnost jedné části neruší celek)?
- [ ] Je řešena součinnost obou stran?
- [ ] Je definováno doručování (adresy, způsob)?
- [ ] Jsou definovány přílohy a tvoří nedílnou součást?

## 13. Křížová kontrola konzistence

- [ ] Souhlasí identifikace stran v záhlaví s podpisovou částí?
- [ ] Jsou termíny (lhůty, data) konzistentní napříč celou smlouvou?
- [ ] Jsou finanční částky konzistentní (zálohy + doplatky = celková cena)?
- [ ] Odpovídají křížové odkazy správným článkům?
- [ ] Jsou definované pojmy používány konzistentně?
- [ ] Neodporují si různé články (např. výpovědní lhůta v čl. 5 vs. čl. 8)?

## 14. Red flags – varovné signály

- [ ] Jednostranné právo druhé strany měnit podmínky
- [ ] Neomezená odpovědnost pouze na jedné straně
- [ ] Příliš vysoké smluvní pokuty bez stropu
- [ ] Automatická prolongace bez možnosti odmítnutí
- [ ] Konkurenční doložka bez kompenzace
- [ ] Exkluzivita bez protiplnění
- [ ] Právo druhé strany na audit bez omezení
- [ ] Vzdání se práv, kterých se dle zákona vzdát nelze (§ 1812 NOZ)
- [ ] Rozhodčí doložka s neznámým rozhodčím soudem
- [ ] Nepřiměřeně dlouhá doba mlčenlivosti (nad 5 let u obchodních smluv)
- [ ] Postoupení smlouvy bez souhlasu
- [ ] Nejasné definice pojmů používaných pro klíčová ustanovení
